P644-P691

回顾I/O

  1. File类:
    1. 目录操作：
      1. 读取目录：listFiles()
      2. 筛选文件：创建实现FilenameFilter接口的类，重写boolean accept()方法，返回endWith()
      3. 创建目录：mkdir();
      4. 删除目录：delete();
  2. 流：
    1. 字节流(重点)
    2. 字符流(重点)

总结：

1. 输入流：Input/Reader

  1. InputStream(字节)和Reader(字符)：（从文件读取数据输入到流中为input和Reader）

    1. InputStream字节：byte

    1. FileInputStream：（可以使用try-with-resource）
      1. 针对文件操作，参数必须为文件路径
      2. 先使用File类声明一个文件对象，再使用file.createNewFile()方法创建文件(createNewFile()必须使用try/catch语句)
      3. 声明一个int类型变量，使用while((c=input.read())!=-1)读取文件内容：=-1 意思就是字符串已到达流末尾而没有可用的字节，JAVA这时候会返回-1值， ！=-1 就是有字节，就读取它

        byte[] b1 = new byte[(int)file.length()];
        int len;
        while((len=fis.read(b1))!=-1)

        System.out.print(new String(b1,0,len));
    2. ByteArrayInputStream:（可以使用try-with-resource）
      1. 针对字符操作，参数必须为字符数组byte[]
    3. BufferedInputStream:(可以使用try-with-resource)
      1. 针对InputStream对象操作，参数必须为InputStream对象(FileInputStream和ByteArrayInputStream)
      2. 缓冲流可以优化I/O

        2. Reader字符：char

    4. FileReader:（可以使用try-with-resource）
      1. 针对文件操作，参数必须为文件路径FilePath
      2. 先声明一个File对象，并使用file.createNewFile()方法创建文件(createNewFile()必须使用try/catch语句)
      3. 读取文件内容：

        int len1;
        while((len1=fr.read())!=-1)
        System.out.print((char) len1);

    5. CharArrayReader：（可以使用try-with-resource）
      1. 具体操作参考ByteArrayInputReader
    6. BufferedReader：（可以使用try-with-resource）
      1. 具体操作参考BufferedInputStream

2. 输出流：Output/Writer

  1. OutputStream(字节)和Writer(字符)：（将流的数据输出到文件为output和Writer)

    1. OutputStream字节：byte

    1. FileOutputStream:（可以使用try-with-resource）
      1. 针对文件操作，参数为filePath
      2. 必须先使用File类声明一个文件对象，并使用file.createNewFile()方法创建文件(createNewFile()必须使用try/catch语句)
      3. 调用output.write(byte byte[]),将字节数组传入
      4. 如果需要读取写入内容，需要再另行实例化FileInputStream，具体操作参考FileInputStream读取内容方法
    2. ByteArrayOutputStrram：（可以使用try-with-resource）（可以通过writeTo写入到FileOutputStream)
      1. 专门针对字节数组
      2. 调用write(byte [])方法写入数据
      3. 若调用了writeTo(fileouputstream)方法，想要读取文件内容：

        fos.close(); //fos先关闭保存，再读取
        byte[] b1 = new byte[(int)file.length()];
        int len;
        while((len=fis.read(b1))!=-1)

        System.out.print(new String(b1,0,len));
    3. BufferedOutputStream:（可以使用try-with-resource）
      1. 专门针对OutputStream对象(FileOutputStream和ByteArrayOutputStrean)
      2. 缓冲流可以优化I/O
      3. 使用try-with-resource实例化OutputStream对象和BufferedOuputStream，传入outputstream对象
      4. 读取内容方法参考FileOutputStreamt

        2. Writer字符：char

    4. FileWriter:（可以使用try-with-resource）
      1. 针对文件操作，参数必须为文件路径FilePath
      2. 先声明一个File对象，并使用file.createNewFile()方法创建文件(createNewFile()必须使用try/catch语句)
      3. 读取文件内容：需要先调用close()关闭保存

        int len1;
        while((len1=fr.read())!=-1)
        System.out.print((char) len1);

    5. CharArrayWriter：（可以使用try-with-resource）
      1. 具体操作参考ByteArrayOutputReader
    6. BufferedWriter：（可以使用try-with-resource）
      1. 具体操作参考BufferedOutputStream

总结字节流和字符流注意事项:

  1. 涉及File类时，字节流读取需要先创建一个byte[]数组来接收文件的内容，再通过new String(byte,0,length)来强制将字节内容转换为字符串
  2. 而字符流可以直接通过((char) c)来输出文件字符内容
  3. 当涉及FileOutputStream/FileWriter写入操作时，如果想读取写入的文件内容，需要先调用close()方法关闭保存

File类：（直接处理文件和文件系统）（获取和操作磁盘文件的信息）（浏览子目录层次）

  1. 构造函数：
    1. File(String directoryPath);directoryPath为路径名
    2. File(String directoryPath,String filename);filename为文件或子目录名
    3. File(File dirObj,String filename);dirObj为指定目录的File对象
    4. File(URI uriobj);uriobj为描述文件的URI对象
  2. 常用方法：
    1. getName()：返回文件名
    2. getParent()：返回父目录名
    3. exists():判断文件是否存在
    4. isFile()：判断调用对象是否为文件类型
    5. isAbsolute():判断调用路径是否为绝对路径
    6. boolean renameTo(File rename):重命名调用对象名称，成功修改返回true，反之false
    7. boolean delete():删除目录，成功返回true
    8. getPath();获取路径
    9. renameTo(File fileobj);

1. 目录操作

  1. 通过listFiles()方法获取该目录下所有的文件名：

  2. 使用FilenameFilter接口过滤得到需要的文件
    使用方式：

    1. 创建一个实现了FilenameFilter接口的类
    2. 定义构造函数参数为(String 需要的文件后缀名)；
    3. 实现boolean accept(File directory,String filename)方法。return name.endswith(后缀名)
    4. 主函数中实例化接口引用，以及实例化实现类的对象，传入后缀名为参数

  3. 创建目录

流类：四个抽象类（InputStream和OutputStream：针对字节流（字节或二进制对象）；Reader和Writer：针对字符流（字符或字符串））

1. 字节流：（顶层：InputStream和OutputStream)

1. FileInputStream类：（针对文件操作）（输入数据到文件）

  1. 构造函数：
    1. FileInputStream(String filePath);
    2. FileInputStream(File fileobj);
  2. 常用方法：
    1. int available():返回当前可读取的输入字节数
    2. void close();
    3. int read();返回代表下一个可以字节的整数
    4. int read(byte buffer[]);读取文件内容到buffer字符数组
    5. void reset();重置文件内容
  3. 实例化：
    1. FileInputStream file = new FileInputStream(File filePath);
    2. FileInputStream file = new FileInputStream(new File(File FilePath));
    3. try-with-resource：try(FileInputStream input = new FileInputStream(File filepath)){}

2. FileOutputStream类：（针对文件操作）（从文件读取数据）

  1. 构造函数：
    1. FileOutputStream(String filePath);
    2. FileOutputStream(File fileobj);
    3. FileOutputStream(String filePath,boolean append);
    4. FileOutputStream(File fileobj,boolean append);
  2. 常用方法：
    1. void write(int b);写入单个int字节
    2. void write(byte buffer[]);写入一个字节数组
  3. 实例化：
    1. FileOutputStream file = new FileOutputStream(File filePath); //
    2. FileOutputStream file = new FileOutputStream(new File(File filePath)); //
    3. try-with-resource：try(FileOutputStream input = new FileOutputStream(File filePath)){}

3. FileInputStream和FileOutputStream综合运用：

4. ByteArrayInputStream类：（输入字节数据到流）

  1. 构造函数：
    1. ByteArrayInputStream(byte array[]);
    2. ByteArrayInputStream(byte array[],int start,int number);从start索引值开始，存入number个字符
  2. 常用方法：
    1. void mark();设置标记
    2. void reset();重置文件内容

5. ByteArrayOutputStream类：（从流输出字节数据）

  1. 构造函数：
    1. ByteArrayOutputStream();32字节缓冲区
    2. ByteArrayOutputStream(int numBytes); 指定numBytes大小的缓冲区
  2. 常用方法：
    1. void write(byte buffer[]);写入一个字符数组到流
    2. void flush();刷新缓冲区
    3. void write(int num);写入单个int类型
    4. void writeTo(FileOutputStream obj);将调用方法的内容写入到obj

6. ByteArrayInputStream和ByteArrayOutputStream综合运用：

7. BufferedInputStream类(缓冲输入字节流)（输入到流）

  1. 构造函数：
    1. BufferedInputStream(InputStream inputstream);参数为InputStream输入字节流的对象
    2. BufferedInputStream(InputStream inputstream,int bufsize);设置缓冲区大小为bufsize
  2. 常用方法：
    1. int available():返回当前可读取的输入字节数
    2. void close();
    3. int read();返回代表下一个可以字节的整数
    4. int read(byte buffer[]);读取文件内容到buffer字符数组
    5. void reset();重置文件内容
    6. void mark(int num);标志num位置
    7. void reset();重置流，回到mark的位置。如果没有mark，返回到开头

8. BufferedOutputStream类：（缓冲输出字节流）（从流输出）

  1. 构造函数：
    1. BufferedOutputStream(OutputStream outputstream);
    2. BufferedOutputStream(OutputStream outputstream,int bufsize)
  2. 常用方法：
    1. void write(int b);写入单个int字节
    2. void write(byte buffer[]);写入一个字节数组
    3. void flush();刷新缓冲区

9. BufferedInput/OutputStream综合运用：

2. 字符流（针对char类型）

1.Reader类：（抽象类）（输入到流）（读取）

方法：

  1. void mark(int num);
  2. int read();读取
  3. int read(char buffer[]); //读取到buffer数组
  4. void reset();

2. Writer类：（从流输出）（写入）

方法：

  1. Writer append(char ch);将字符添加到输出流的末尾
  2. void writer(int ch)；写入单个int字符
  3. void writer(String str);写入字符串

3. FileReader类：（从文件读取数据）（针对char类型）

  1. 构造函数：
    1. FileReader(String filePath);
    2. FileReader(File fileobj);
  2. 方法：同Reader类

4. FileWriter类：（写入数据到文件）（针对char类型）

  1. 构造函数：
    1. FileWriter(String filePath)
    2. FileWriter(String filePath,boolean append)
    3. FileWriter(File fileobj)
    4. FileWriter(File fileobj,boolean append);boolean append为true，加载文件
  2. FileWriter类对象实例时会自动创建目标文件
  3. 方法：同Writer类

  1. 构造函数：
    1. CharArrayReader(char array[]);
    2. CharArrayReader(char array[],int start，int numchars);根据字符数组的子集创建Reader对象，从start索引开始读取，一共读取numchars个字符

7. CharArrayWriter类：（输入流）（将流中的数据写入）

  1. 构造函数：
    1. CharArrayWriter();
    2. CharArrayWriter(int numchars);指定缓冲区大小

  1. 构造函数：
    1. BufferedWriter(Writer outputstream);
