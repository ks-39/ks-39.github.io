P235-P257

今日内容：

  1. 多线程：
    1. 基本线程方法:getName()获取线程名称、getPriority()获取线程优先级、isAlive()确定线程是否运行、join()等待线程终止、run()线程入口、sleep()等待时间(挂起)、start()通过调用现成run()方法启动线程。
    2. 主线程:即main函数,可以通过Thread MainThread = Thread.currentThread();读取主线程信息(线程组，优先级，线程名)
    3. 创建子线程(重点)：
      1. Thread类：
        1. 先创建一个继承了Thread类的类：class A extends Thread(){}
        2. 在类的构造函数中调用super(线程名)创建子线程，并调用start()方法启动run()方法
        3. 书写run()方法：public void run(){},如需要调用
          Thread.sleep()静态方法，需要搭配try/catch语句
        4. 在main函数中声明类：new A();调用构造函数，创建子线程
      2. Runnable接口：
        1. 创建一个实现了Runnable接口的类：class A implements Runnable{}
        2. 声明Thread类的对象：Thread t;
        3. 在类的构造函数中创建子线程：t = new Thread(this,”线程名”);this是指定start()方法调用该类中的run()方法。调用start()方法：t.start();
        4. 书写run()方法：public void run(){},如需调用Thread.sleep()静态方法，需要搭配try/catch语句
        5. 在main函数中声明类：new A();调用构造函数，创建子线程
    4. 线程优先级：
      1. setPriority()设置优先级
      2. getPriority()获取优先级
  2. 同步：当一个线程正在读取该数据结构时，必须阻止另一个线程向该数据结构写入数据
    1. 同步方法：使用synchronized修饰方法：synchronized void meth(){}
    2. synchronized语句：使用

主程序：main方法

  1. 其它线程都是从主线程产生
  2. 主线程最后执行结束，因为需要执行关闭动作
  3. 主线程是系统自动创建的，可以通过static Thread currentThread()方法获取主线程的引用。

Thread类的一些方法：getName()获取线程名称、getPriority()获取线程优先级、isAlive()确定线程是否运行、join()等待线程终止、run()线程入口、sleep()等待时间(挂起)、start()方法(启动run()方法启动线程)

创建子线程

  1. 实现Runnable接口
  2. 拓展Thread类

1. 实现Runnable接口

总结实现Runnable接口方式：

  1. 创建Runnable接口的类：class classname implements Runnable
  2. 在类中声明一个Thread类引用：Thread t;使用Runnable接口需要声明类引用。
  3. 书写主函数，在主函数中调用类的构造函数来新建线程:new NewThreead();（注意，main方法就是主线程）
  4. 在构造函数中将线程引用指向新的Thread类对象：t = new Thread(),在使用start()方法调用run()方法：t.start();
  5. 创建run()方法：public void run()。在run()方法中定义子线程的代码。
  6. 补全主线程。
    注意：使用sleep()方法需要try/catch语句。

2. 拓展Thread类

总结拓展Thread类的步骤：

  1. 创建一个拓展了Thread类的类：class NewThread extends Thread{}
  2. 在main函数中创建该类的实例：new NewThread();调用类的构造函数
  3. 在NewThread类中书写构造函数，在构造函数中调用super()方法(因为NewThread类是继承自Thread类的)创建一个新线程：super(“创建新线程”);然后再直接调用start()方法启动子线程：start().
  4. 创建run()方法：public void run(){},在run()方法中定义子线程。
  5. 补全主线程。
    注意：使用sleep()方法需要try/catch语句。

创建多个线程:(继承Thread类)

需要注意的是Thread.sleep()，sleep是Thread类中的一个静态方法，调用时，只需表明在Thread类中调用（直接Thread.sleep()），而不需要使用对象引用（t.sleep()）。最好的调用方法就是 Thread.sleep()。主线程（main函数）需要设定足够长的延迟时间来确保它是最后执行完毕的。

isAlive()和join()

  1. isAlive()方法:final boolean isAlive()
  2. join()方法等待线程结束:final void join()

使用实现Runnable接口的方式时，想要调用Thread类的方法（除了静态方法）都必须要先通过NewThread类的对象（n1、n2、n3）引用Thread类的引用（t）再通过Thread类的引用来调用方法,如：n1.t.isAlive()和n1.t.join()。对于sleep()静态方法，直接指定Thread类来调用（Thread.sleep())。

同步

监视器概念：当一个线程正在读取该数据结构时，必须阻止另一个线程向该数据结构写入数据。一旦某个线程进入监视器，其他线程必须等待，直到该线程退出监视器。每个对象都有自己的隐式监视器。

1. 使用同步方法：调用使用synchronized关键字修饰方法(暂无方法做到顺序输出)

一旦线程进入了一个实例的同步方法，其他线程就不能再进入相同实例的同步方法

wait()方法通知调用线程进入休眠
notify()方法唤醒调用相同对象的wait()方法的线程
notifyAll()方法唤醒调用相同对象的wait()方法的所有线程
