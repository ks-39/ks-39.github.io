I/O

  1. java通过流执行I/O。流是一种抽象，要么产生信息，要么使用信息。（java.io包）
  2. 字节流：读取和写入二进制时使用字节流，使用Unicode编码
  3. 字节流类：两个顶级抽象类：InputStream和OutputStream
    java.io包中的字节流类：

    抽象类InputStream和OutputStream定义了两个重要方法read()和write()。
  4. 字符流类：两个顶级抽象类：Reader和Writer。
    定义了两个重要方法read()和write()
  5. 预定义流：System类包含三个流变量：in、out和err。被默认定义为public、static和final，可以被其它任何代码访问。
    System.out引用标准的输出流，默认情况是控制台，是InputStream的对象
    System.in引用标准的输入流，默认情况是键盘，是OutputStream的对象
    System.err引用标准的错误流，默认情况是控制台，是OutputStream的对象

1. 读取控制台输入（基于字符）

  1. 创建一个与键盘连接的BufferedReader对象：BufferedReader br = new BufferedReader(new InputStreamReader(System.in));其中InputStreamReader将System.in读取的字节转换为字符，然后作为参数传给BufferedReader的构造函数。

NIO系统：（构建于缓冲区和通道）（缓冲区用于容纳数据，通道打开连接）（先连接通道和缓冲区，再输入输出数据）

1. 缓冲区(Buffer)：（抽象类）（Buffer的重要属性：capacity(容量)、limit(上限)、position(位置)、mark(标记)

  1. Buffer本质上是一块内存区，可以存储和输出数据

  2. 涵盖了Java的基本数据类型：ByteBuffer、CharBuffer、ShortBuffer、IntBuffer、FloatBuffer、DoubleBuffer、LongBuffer。

  3. 基本属性理解：

    1. ByteBuffer buffer = ByteBuffer.allocate(1024);大小为1024个字节(byte),position=0；limit=1024；capacity=1024；

    2. 写入字节数据：buffer.put(“abcd”.getBytes()); //注意转换为字节类型,此时position为4，limit=1024，capacity=1024； //position可以理解为记事本光标位置

    3. 读取字节数据：buffer.flip(); filp()源码

      //读取”abcd”字节，此时position=0(回到开头)，limit=4(缓冲区读取了4个字节数据，此时缓冲区指针位于4)；capacity=1024

    4. 清空缓冲区：buffer.clear();指针归0，limit恢复初始大小1024，capacity不变1024；

  4. 写入数据：

    1. 从Channel写入数据到Buffer：
      1. int byteWrite = inChannel.write(buffer);
    2. 使用put方法写数据Buffer：
      1. buffer.put(123/“…”.getByte())：

  5. 读取数据：

    1. 从Channel读取数据到Buffer：
      1. int byteRead = inChannel.read(buffer);
    2. 使用get方法从Buffer读取数据：
      1. buffer.get();

  2. 写入数据：
