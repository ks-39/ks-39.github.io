继承(重点)

1. 子类使用extends关键字继承超类

2. 就算子类能继承超类的所有成员，也无法继承超类中被声明为私有的成员。例如：

3. 使用继承可以拓展类的成员

一旦创建一个定义对象通用（不是private）的超类，就可以用子类继承这个超类，每个子类在增加自己特有的属性(成员)，这就是继承的本质。

4. 超类变量可以引用子类对象

当指向子类对象的引用(weightbox)赋值给超类的引用变量(子类和超类共有的width、height、depth)时，只能访问子类与超类共有的部分(也就是只能访问width、height和depth)。因为超类不知道子类添加了什么成员。

5. 超类引用变量可以指向子类对象

在main方法中，声明了a、b、c三个对象，还声明了一个A的引用变量r，当r指向A子类的对象后，就可以使用r引用变量调用子类中的方法

6. 多级继承

假设有A、B、C三个类，C是B的子类，B又是A的子类，那么C就拥有B和A的所有特征。

具体步骤形如一级继承，需要注意在super方法传入参数必须要加上Boxadd定义的weight变量，才能调用到Boxadd类中相对应的构造函数。

String类

  2. String类方法：

    1. equals()方法测试两个字符串是否相等：boolean equals(secondStr)
    2. length()方法获取字符串长度：int length(mystring) 或者 obj.length()
    3. charAt()方法获取字符串中指定索引处的字符：char charAt(index),String数组没有charAt方法（因为数组都没有charAt方法）

varargs：可变长度参数：当参数的具体数量不能确定时，一般将参数放入数组中，然后将数组传递给方法。可变长度参数(type …val)必须放在最后面声明
