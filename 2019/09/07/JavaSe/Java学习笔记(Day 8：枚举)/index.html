枚举

最简单的枚举是一系列有名称的常量，每一个枚举常量都是枚举的对象

1. 创建枚举：enum关键字

  1. 标识符basketball等被称为枚举常量。被隐式声明为BALL类的公有、静态final成员。
  2. 定义类枚举后，创建枚举类型的变量：BALL ba；因为ba是BALL类型，只能被赋值为在BALL中定义的枚举常量：ba = BALL.football；
  3. 可以使用“==”来比较枚举常量。if(ba==BALL.football)
  4. 枚举值可以控制switch语句：switch(ba){},case语句使用的常量必须是同一个枚举中的常量:case basketball

2. values()和valueOf()

1、values:
public statci BALL [] values()
values()方法返回一个包含枚举常量列表的数组
2、valueOf：
public static BALL valueOf(String str)
valueOf()方法返回与传递到参数str的字符串相对应的枚举常量。

3. 枚举是类类型

枚举可以定义构造函数（也就意味着可以重载）、可以定义实例变量和方法、甚至可以实现接口。

在main（）方法中声明变量ob时，对每个特定的常量调用了BALL的构造函数一次。

4. 枚举重载构造函数

枚举不能继承其他类（自动继承Enum类，所以不能继承其他类），也不能是超类

5. 枚举的ordinal()方法、equals()、equalsTo()

1、ordinal()方法：BALL.x.ordinal():返回调用枚举常量的序数值
2、equals()方法：if(ob1.equals(ob2)):比较两个枚举常量的相等性
3、compareTo()方法：if(ob1.compareTo(ob2)>0):比较两个枚举常量序数值的大小。

类型封装器

1、Character是char类型的封装器,构造函数：Character(char ch)
获取char值：char charValue()
2、Boolean是布尔值的封装器，构造函数：Boolean（boolean boolvalue）或者Boolean（String boolstring）
3、Integer是所有数值类型封装器：构造函数：Integer（int num）
获取返回值：type typeValues();
可以通过封装器对象传递给println方法：Integer ob = new Integer(100);int i = ob.intValues();
