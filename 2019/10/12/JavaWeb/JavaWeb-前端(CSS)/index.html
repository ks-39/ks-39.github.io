2. CSS语法

  1. 格式:
    选择器{ //css控制的标签

    属性名1:属性值;     //使用;隔开属性
....;
属性名n:属性值

    }

  2. 选择器:

    1. 基本选择器

      1. id选择器:#id{}。优先级最高

      2. 元素选择器:元素{}。多所有的元素进行控制。优先级最低

      3. 类选择器:.class{}。对所有具有相同class属性值的元素进行控制。优先级中等

    2. 拓展选择器

      1. 选择所有元素:*{}

      2. 并集选择器:选1,选2{} //同时控制选1和选2

      3. 子选择器:选1 选2{} //控制选1下的选2

      4. 父选择器:选1 > 选2 //筛选选2的父元素选1

      5. 属性选择器:元素名[属性=”属性值”]{}

      6. 伪类选择器:元素:状态{}

        1. 状态:
          1. link:初始化
          2. visited:被访问过
          3. active:正在访问
          4. hover:鼠标悬浮状态
