内容:

  1. JavaScript = ECMAScript + (BOM+DOM)：（控制页面元素，让页面拥有动态效果）
  2. BOM和DOM的关系
    1. BOM是浏览器对象模型，用来获取或设置浏览器的属性、行为，例如：新建窗口、获取屏幕分辨率、浏览器版本号等。
    2. DOM是文档对象模型，用来获取或设置文档中标签的属性，例如获取或者设置input表单的value值。
    3. BOM的内容不多，主要还是DOM。由于DOM的操作对象是文档（Document），所以dom和浏览器没有直接关系。
    4. BOM六大模块:frame，history，navigator，document，location，screen。其中document就是我们所说的DOM。document与其他不同的地方在于，其他四大对象是对浏览器操作。而document是对浏览器中显示的网页做的操作，所以叫做DOM
  3. BOM的核心是window对象(即浏览器页面),其六大核心模块为document(即DOM)/frane/history/location/navigator/screen.
    1. window弹窗类效果:alert()/confirm()/prompt()
    2. window开关页面:close()/open()
    3. window定时器:setTimeout()/clearTimeout()/setInterval()/clearInterval()
    4. Location.href:浏览器地址栏网页地址
  4. DOM(即Document对象,控制HTML文档内容)
    1. 查找HTML元素:getElementById()/getElementByTagName()/getElementByClassName()/getElementByName()
    2. 创建对象:createAttribute()/createElement()
    3. 改变HTML内容:document.getElementById(id).innerHTML() = “new text”; //仅支持改变文本内容
    4. 改变CSS内容:document.getElementById(id).style.attrubute = “new attribute”; //改变属性值
