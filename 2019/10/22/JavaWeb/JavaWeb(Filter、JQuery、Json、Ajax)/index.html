    1. 拦截路径:

      1. 具体资源路径拦截:/index.jsp
      2. 拦截目录:/user/*
      3. 后缀名拦截:*.jsp
      4. 拦截所有资源：/*

    2. 拦截方式:

      1. 注解:设置dispatcherTypes属性

        1. REQUEST:默认值,浏览器直接请求资源
        2. FORWARD:转发访问

      2. web.xml

  5. Filter链(配置多个过滤器)

    1. 执行顺序:
      1. F1 start
      2. F2 start
      3. 资源执行
      4. F2 end
      5. F1 end
    2. 先后顺序:
      1. 注解配置:字符串小的先执行
      2. xml配置:先配置先执行

  6. 案例1:登陆验证：

    1. 需求:
      1. 当用户访问服务器其他资源时,进行拦截,验证用户是否已经登录
      2. 如果登录了,直接放行
      3. 如果没有登录,跳转至登录页面
    2. 思路:
      1. 获取资源请求路径:request.getRequestURI()
      2. 判断是否包含登录相关资源(也就是用户是否是在访问login.jsp页面):uri.contains(“/login.jsp”)….
      3. 包含则放行继续操作,不包含则判断用户是否已经登录(获取session的user属性):requestgetSession.getAttribute(“user”);读取session中的user信息.如果user不为空则代表已经登录,如果为空则重定向到login.jsp页面

  7. 案例2:过滤敏感词

    1. 思路:
      1. 对request对象的getParameter()方法进行增强,生成一个新的request
      2. 放行,将新request对象传入
    2. 增强对象的功能:通过代理模式的动态代理实现
      1. 代理模式增强方法:(与真实对象实现相同的接口)
        1. Proxy.newProxyInstance(obj.getClass().getClassLoader(),obj.getClass().getInterfaces(),new InvocationHandler())
      2. 实现步骤:
        1. 代理对象和真实对象实现相同的接口
        2. 代理对象 = Proxy.new ProxyInstance()
        3. 使用代理对象调用方法
        4. 增强方法

  3. JQuery选择器:

    1. 基本选择器:
      1. 标签(元素):$(“标签名”)
      2. id:$(“#id值”)
      3. 类:$(“.class值”)
    2. 层级选择器(元素标签):
      1. 后代:$(“A B”):A标签的所有B元素:(所有后代元素)(多层)
      2. 子选择:$(“A > B”):A标签的所有B标签的子元素：(一层)
    3. 属性选择器(属性值):
      1. 属性名:$(“A[属性]”)
      2. 属性:$(“A[属性=值]”)
      3. 符合属性:$(“A[属性=值][]..)
    4. 表单过滤选择器:
      1. 可用元素过滤:$(“A[属性=值]:enable”)
      2. 不可用元素过滤:$(“A[属性=值]:disable”)
      3. 单/复选框选中选择器:$(“A[属性=值]:checked”)
      4. 下拉列表选中:$(“A[属性=值]:selected”)

  4. DOM操作:

    1. 内容操作:

      1. html():获取/设置元素标签体内容
      2. text():获取/设置元素标签体纯文本内容
      3. val():获取/设置元素的value属性值

    2. 属性操作:

      1. 通用属性
        1. attr():获取/设置元素的属性
        2. removeAttr():删除属性
        3. prop():获取/设置元素的固有属性
        4. removeProp():删除固有属性
      2. class属性
        1. addClass():添加class属性
        2. removeClass():删除class属性
        3. toggleClass():切换class属性

    3. CRUD:

      1. 对象1.append(对象2):将对象2追加到对象1末尾
      2. 对象1.prepend(对象2):将对象2追加到对象1开头
      3. 对象1.appendTo(对象2):将对象1追加到对象2末尾
      4. 对象1.prepend(对象2):将对象1追加到对象2开头
      5. 对象1.after(对象2):将1添加到2后
      6. 对象1.before(对象2):将1添加到2前
      7. 对象1.insertAfter(对象2):将2添加到1后
      8. 对象1.insertBefore(对象2):将2添加到1前
      9. 对象.remove():删除对象
