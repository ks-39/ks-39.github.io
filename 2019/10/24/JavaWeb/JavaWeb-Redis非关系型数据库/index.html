关系型和非关系型数据库

  1. 关系型数据库:MySQL、Oracle
    1. 数据之间有关联关系
    2. 数据存储在硬盘文件上
    3. 数据载体是表
  2. 非关系型数据库:Redis
    1. 数据之间没有关联关系
    2. 数据存储在内存中
    3. 数据格式为键值对

Redis

  1. 命令操作
    1. 数据结构:key是字符串,value有5种数据结构
      1. String:
      2. hash:map格式
      3. list:列表
      4. set:集合
      5. sortedset:有序集合
    2. String:
      1. 存储:set key value
      2. 获取:get key
      3. 删除:del key
    3. hash:
      1. 存储:hset key field value
      2. 获取:hget key field
      3. 删除:hdel key field
    4. list:(允许重复元素)
      1. 存储:
        1. lpush key value:加入列表左边
        2. rpush key value:加入列表右边
      2. 获取:
        1. lrange key start end:获取start到end之间的元素
      3. 删除:
        1. lpop key:返回并删除最左边的元素
        2. rpop key:返回并删除最右边的元素
    5. set:(不允许重复元素)
      1. 存储:sadd key value
      2. 获取集合所有元素:smenbers key
      3. 删除:srem key value
    6. sortedset:(有序集合)
      1. 存储:zadd key score value;
      2. 获取:zrange key start end;获取start到end之间的元素
      3. 删除:zrem key value
    7. 通用操作:
      1. keys *:获取所有键
      2. type key:获取键的value类型
      3. del key
