总结：

  1. 概念：Java Database Connectivity Java。Java语言操作数据库
  2. 本质：一套操作所有关系型数据库的规则(接口)，驱动jar包的实现类来操作数据库。包括MySQL版本、Oracle版本和DB2版本
  3. 基本步骤：
    1. 导入驱动jar包：mysql-connector-java
    2. 注册驱动：Class.forName(“com.mysql.cj.jdbc.Driver”);
    3. 定义sql语句：
      1. Statement: String sql = “….”;
      2. PrepareStatement: String sql = “….. = ? …. = ? …”; //使用?占位符替代参数
    4. 获取数据库连接对象Connection：Connection con = DriverManager.getConnection(“jdbc:mysql://localhost:3306/db1?serverTimezone=GMT”, “root”, “password”);
    5. 获取执行sql语句的对象Statement:Statement sta = con.createStatement();/PrepareStatement psta = con.PrepareStatement();
      1. 使用PrepareStatement()：给?赋值
        psta.set…();
    6. 执行sql语句,接收返回结果
      int result = sta.executeUpdate(sql);
      ResultSet rs = sta.executeQuery(sql);
      int result = psta.executeUpdate(); //PrepareStatement()不需要传入sql
      ResultSet rs = psta.executeQuery();
    7. 处理结果：while(rs.next()){}
    8. 释放资源：con.close();sta.close();rs.close();
  4. 使用JDBC工具类Utils:
    1. 新建配置文件(jdbc.properties),写入url、user、password和driver。（properties配置文件不能使用空格和””)
    2. 创建Utils类声明url、user、password和driver为私有静态变量
    3. 创建静态代码块
      1. 创建Properties类,使用load方法获取src下的配置文件：Properties pro = new Properties();
      2. 声明ClassLoader类对象:ClassLoader load = JDBCUtils.class.getClassLoader();
      3. 使用InputStream流接收getResourceAsStream(“jdbc.properties”)返回的路径
      4. pro.load(inputstream);读取properties配置文件
      5. 给私有变量赋值:url = pro.getProperty(“url”);
      6. 注册驱动：Class.forName(driver); //在静态代码块中注册驱动
    4. 创建获取连接对象方法:public static Connection getConnection(){
      return DriverManager.getConnection(url,user,password); //返回类型为Connection类对象
      }
    5. 创建释放资源方法:public static void close(Connection con,Statement sta,….){}
    6. 具体实现类：
      1. 获取链接和注册驱动：
        con.JDBCUtils.getConnection();
      2. 开启事务
      3. 定义sql语句
      4. 获取执行sql的对象PrepareStatement psta;
      5. 设置?参数
      6. 执行sql
      7. 提交事务
      8. 回滚事务
      9. 释放资源：JDBCUtils.close(con,psta…);
  5. C3P0数据库连接池
    1. 导入jar包add as library:c3p0-0.9.5.2.jar/mchange-commons-java-0.2.12.jar
    2. 定义配置文件：
      1. 名称：c3p0.properties 或者 c3p0-config.xml //properties格式:不能使用双引号和空格;
      2. 路径：配置文件放在src目录下
    3. 创建核心对象ComboPooledDateSource:DataSource ds = new ComboPooledDataSource();
    4. 获取连接getConnection():Connection con = ds.getConnection();
  6. Druid数据库连接池（使用DruidUtils工具类）
    1. 导入jar包add as library:druid-1.1.16.jar
    2. 定义配置文件:
      1. 名称：druid.properties //不能使用空格和双引号,可以为任意名称
      2. 路径：可以在任意路径
    3. 静态代码块加载配置文件：
      1. 创建Properties类：Properties pro = new Properties();
      2. 使用InputStream对象接收DruidTest.class.getClassLoader().getResourceAsStream()返回的路径;
      3. pro.load(input);读取配置文件参数
      4. 获取数据库连接池对象：通过工厂类(DruidDataSourceFactory)获取
        DataSource ds = DruidDataSourceFactory.createDataSource(pro); //DataSource读取xing
    4. 获取连接对象方法：
      public static Connection getConnection(){
      return ds.getConnection();
      }
    5. 释放资源方法
    6. 获取连接池方法：
      public static DataSource getDataSource(){
      return ds;
      }
    7. main方法：
      1. 获取链接和注册驱动：
        con.DruidUtils.getConnection();
      2. 开启事务
      3. 定义sql语句
      4. 获取执行sql的对象PrepareStatement psta;
      5. 设置?参数
      6. 执行sql
      7. 提交事务
      8. 回滚事务
      9. 释放资源：DruidUtils.close(con,psta…);

入门操作案例：

拆解

1. DriverManager类：（驱动管理对象）

  1. 功能：
    1. 注册驱动:（static void registerDriver(Driver driver)静态方法）（可有可无）
      1. 代码实现：
        Class.forNaem(“com.mysql.cj.jdbc.Driver”); //在com.mysql.cj.jdbc.Driver类中存在静态代码块
    2. 获取数据库连接：（static Connection getConnection(String url,String user,String password);
      1. 代码实现：（mysql)
        DriverManager.getConnection(“jdbc:mysql://localhost:3306/db1?serverTimezone=GMT”,”root”,”password”); //serverTimezone=GMT设置时区，mysql默认端口3306,可以省略

2. Connection类：（数据库连接对象）

  1. 功能：
    1. 获取执行sql的对象
      1. Statement createStatement(); //不推荐
      2. PrepareStatement prepareStatement(String sql); //需要给?占位符赋值
    2. 管理事务
      1. 开启事务:con.setAutoCommit(boolean autoCommit); //false 开启,
      2. 提交事务:con.commit();
      3. 回滚事务:con.rollback();

3. Statement类：（执行sql语句对象）

  1. 执行sql语句
    1. int executeUpdate(String sql); //执行DML(增删改)，返回值为影响的行数。
    2. ResultSet executeQuery(String sql); //执行DQL(select)语句
  2. DML语句（增删改）：
    1. account表添加一条记录
      String sql = “insert into account values(null,’王五’,3000”;
    2. account表修改记录
      String sql = “update account set balance = 1000 where id = 3”;
    3. account表删除一条记录
      String sql = “delete from account where id = 3”;
  3. DDL语句（create表）
    1. 创建表
      String sql = “create table info (id int, name varchar(20))”;

4. ResultSet类：（结果集对象,封装查询结果）

  1. 执行select语句：

    1. String sql = “select * from account”;

  2. 使用executeQuery()执行select语句

  3. 普通查询方式

5. PreparedStatement类：

  1. sql注入问题:在sql语句拼接时，有sql关键字参与，
  2. 解决sql注入步骤：
    1. 导入驱动jar包：mysql-connector-java
    2. 注册驱动：Class.forName(“com.mysql.cj.jdbc.Driver”);
    3. 定义sql语句：String sql = “…”; 参数使用?作为占位符.如：”select * from user where username = ? and password = ?”;
    4. 获取数据库连接对象Connection：Connection con = DriverManager.getConnection(“jdbc:mysql://localhost:3306/db1?serverTimezone=GMT”, “root”, “password”);
    5. 获取执行sql语句的对象PrepareStatement:PreparedStatement psta = Connection.prepareStatement(String sql);
    6. 给?赋值:setusername(参数1,参数2); //参数1为?位置编号(从1开始),参数2为?的值
    7. 执行sql语句,接收返回结果,不需要传递sql
      int result = psta.executeUpdate();
      ResultSet rs = psta.executeQuery();
    8. 处理结果：while(rs.next()){}
    9. 释放资源：con.close();sta.close();rs.close();
