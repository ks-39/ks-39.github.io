P1-P102

1. 作用域：

  1. 由{ }包裹起来的区域，在此区域内初始化的变量只能在此处被访问。
  2. 如果是for循环语句作用域，变量在作用域内被访问了，那么在每次循环时都会被重新初始化。
  3. 就算是在作用域内也不能声明与外层代码块有相同名称的变量。

2. 自动类型提升

  1. 如果计算过程的结果超出了byte(8)、short(16)、char(2字节，16bit)类型的范围，会自动提升为int类型（32）
  2. float转换为int会发生截尾现象，即小数部分被舍去。

3. 数组(重点)

  1. 声明一维数组：type var[] 或者 type [] var
  2. 使用new运算符分配数组：array-var = new type[size];type指定了分配的数据的类型; size指定了数组中元素的数量，即var.length。使用new分配的数组，对于int类型，元素自动初始化为0；对于布尔类型，元素自动化为false；对于引用类型，元素自动化为null（空）。
  3. 实例：MonthDays = new int[12];分配了12个元素的int一维数组，并且都为0
  4. 回顾步骤：首先必须声明一个数组类型的变量，如：int var[];其次，必须使用new分配该数组的内存并赋值给数组变量。如：var = new int[size];
  5. 分配数组后可以通过索引的方法来访问数组中的元素。索引值从0开始。常用初始化方式：int var[] = new int[size];或在声明的同时直接初始化：int var[] = {0,1,2,3,4};

3. while语句

  1. 一般形式：
    while(condition){
    // body of loop
    }
    condition可以是任何布尔表达式。只要表达式为true，就执行循环。为false则跳转至下一行代码。同样的，如果需要执行的语句只有一条，则不需要{ }。
  2. while循环体可以为空。

4. do-while语句

  1. 一般形式：

  2. do-while循环每次迭代首先执行循环体，然后对条件表达式进行求值，如果表达式为true，继续循环，否则终止循环。condition必须是布尔表达式。

5. for语句（一般只用于控制循环，在其他地方不使用）

  1. 一般形式：
    for(initialization(初始化);condition(循环条件);iteration) {
    // body
    }
  2. 在for循环内部声明循环控制变量

for循环控制变量通常只用于控制循环，在其他地方不使用，当for语句结束时，变量的作用域也随之结束。也就是说，在for循环内部声明的变量，只有在for循环内部有作用，在for循环外，该变量是不存在的，如果需要在程序的其他地方使用循环控制变量，就不能在for循环内部声明它

6. for-each（重点）

  1. 一般形式：
    for(type itr-var : collection)

    statement-block

    type指定了类型，itr-var为迭代变量的名称，用于接收来自集合的元素，从开始到结束，每次接收一个，collection指定了要遍历的集合(数组或者是由集合框架定义的集合类型)，迭代的元素存储在itr-var中，因此itr-var类型必须与集合中保存的元素类型相同（或兼容）。

for-each算法：二分查找

使用选择排序法和二分查找法查找元素:

  1. 先对数组进行排序：数字前一位和后一位作比较，通过一个中间变量来交换索引
  2. 打印输出检验
  3. 定义start、middle、end、count变量，start=0;middle=(start+end)/2;end=数组index,count=0。
  4. 当number!=middle时:
    1. 如果number>middle:代表number在middle到end这一半,start从middle开始,middle再=(start+end)/2,count+1。
    2. 如果number < middle:代表number在start到middle这一半,end到middle为止,middle=(start+end)/2,count+1
    3. 当count>数组的一半时(count>a.length)，代表在这一半的数组中查无此数;反之则代表number在这一半数组中。
