注意事项：

  1. Linux命令区分大小写
  2. Linux所有内容以文件形式保存，包括硬件。如：硬盘文件:/dev/sd[a-p]
  3. “Tab”自动补全、
  4. Linux没有后缀
    1. 压缩文件:”.gz”、”.bz2”、”.tar.bz2”、”.tgz”、
    2. 二进制:”.rpm”
    3. 网页文件:”.html”
    4. 脚本:”.sh”
    5. 配置文件:”.conf”
  5. Windows程序不能直接使用在Linux
  6. Linux的存储设备必须先挂载
  7. 服务器不允许关机,只能重启
  8. 重启时应该关闭服务

Linux目录

  1. 系统命令:s指代超级用户
    1. /bin/:
    2. /sbin/:
    3. /usr/bin/:
    4. /usr/sbin/:
  2. /boot/:系统启动目录,保存系统启动相关文件
  3. /dev/:设备文件
  4. /etc/:配置文件
  5. /home/:用户目录
  6. /lib/:数据库
  7. /lost+found/:文件修复
  8. /media/、/mnt/、/misc/:挂载目录
  9. /proc/、/sys/:虚拟文件系统,存储在内存,重启即初始化
  10. /root/:

Linux命令:命令 [-选项] [参数]

1. 目录处理命令

1. 查看目录命令:ls

  1. 语法:
    1. -a:显示所有文件，包括隐藏
    2. -l:详细信息显示
    3. -d:查看目录属性
    4. -h:人性化显示
    5. -i:查看id
    6. 例子:ls -a -lh
  2. rw-r–r–:u/g/o：所有者/所属组/其他人
  3. r/w/x:读/写/执行

2. 创建目录命令:mkdir

  1. 语法:
    1. -p:递归创建,创建多级目录
    2. mkdir //0 //1:创建多个目录
    3. 例1:mkdir -p /tmp/test0
    4. 例2:mkdir /tmp/test1 /tmp/test2

3. 查看当前位置命令:pwd

4. 跳转目录命令:cd

  1. 语法:
    1. cd ..:回到上一级目录
    2. cd /**/1:回到指定目录

5. 删除目录命令:rmdir

  1. 语法:
    1. rmdir /**/test:删除空目录

6. 复制目录命令:cp

  1. 语法:
    1. 复制文件:cp /test1/1.txt /test2
    2. 复制目录:cp -r /test1 /home
    3. 复制多个文件:cp /test1/1.txt /test1/2.txt /test2
    4. 保留文件属性:cp -p /test1/1.txt /test2
    5. 复制的同时改名:cp -r /test1/1.txt /test2/2.txt

7. 剪切目录命令:mv

  1. 语法:
    1. 改名:mv /test1/1.txt /test1/2.txt
    2. 其余参考复制命令

8. 删除文件/目录命令:rm

  1. 语法:
    1. 删除文件:rm 1.txt
    2. 删除目录:rm -rf /**/test1

2. 文件处理命令:

1. 创建文件命令:touch

  1. 语法:
    1. 在指定目录下创建文件:touch /test/1.txt

2. 查看小内容文件命令:cat

  1. 语法:
    1. 显示行号:cat -n /test/1.txt

3. 查看大内容文件命令:

  1. more

    1. more /test/1.txt
    2. 翻页:space/f
    3. 换行:enter
    4. 退出:q/Q

  2. less

    1. 可使用PU/PD,方向键。

4. 链接:

  1. ln
    1. 软链接:ln -s /test/1.soft
    2. 硬链接:ln /test/1.hard

3. 权限管理命令:

1. 更改权限命令:chmod

  1. 语法:u/g (+-) r/w/x
    1. 更改文件或目录权限:chmod u+x /test/1.txt

4. 文件搜索命令:

1. 文件搜索命令:find

  1. 语法:
    1. find /test -name 1:搜索test目录下名为1的文件
    2. find /test -name 1:模糊搜索
    3. find /test -size (+-)204800:搜索大于/小于/等于100MB的文件
    4. find /test -groupname:搜索用户组文件
    5. find /test -type f/d/l:搜索file/目录/链接

2. 资料库搜索命令:locate.

  1. 新文件无法使用
  2. updatedb先更新
  3. tmp目录下无法使用

3. 搜索命令目录:which

4. 文件内容中搜索:grep

  1. 语法:
    1. 不区分大小写:grep -i

5. 用户管理

  1. -c:描述

  2. -u:UID

  3. -G:组名

    1. 添加用户:useradd

  4. 语法

    1. useradd -u 500 username:指定UID

  5. 更改密码:passwd username

  6. 查看用户:who/w

  7. 查看服务器运行时间:uptime

  8. 修改用户信息:usermod

  9. 删除用户:userdel -r username

  10. 查看当前用户:whoami

  11. 登录root:su root

  12. 添加用户组:groupadd

  13. 删除用户组:groupdel

6. 压缩解压命令:

1. gz

  1. 压缩:gzip test

  2. 解压:gunzip test

2. tar

  1. 压缩:tar -zcf 1.tar /test

  2. 解压:tar -zxf 1.tar

3. zip

  1. 压缩:zip 1.zip test

  2. 解压:unzip 1.zip

  3. 解压:bzip -k 1.zip

7. 网络命令:

1. 给指定用户发送信息:write

  1. 语法:write 用户名。以Ctrl+D结束

2. 广播信息:wall

  1. 语法:wall 信息。

3. 检验网络:ping

  1. 语法:ping ip地址。

4. 检验网络配置:ifconfig

5. 设置Linux的IP地址:ifconfig eth0 192.168.xxx.xxx

8. 关机重启

  1. shutdown -c:取消关机命令

  2. shutdown -h:关机

  3. shutdown -r:重启

  4. logout:操作之后退出

9. Vim：

  1. vim filename进入命令模式

  2. : 进入编辑模式

  3. aA/iI/oO:在光标/行尾输入

  4. set nu:显示行号

  5. : wq 保存退出

  2. 查看别名:alias

  1. ;:多命令执行.

  2. *:任意个字符

Shell编程

1. cut命令

  1. -f:按字段筛选,默认不加参数-d时,分隔符为‘tab’键。如 -d ‘ ‘ -f 2 表示用空格作为分隔符，提取第二列的内容

  2. -d:自定义分隔符,默认是制表符,例如 -d ‘ ‘ 表示用空格作为分隔符,-d : 表示用冒号作为分隔符

2. printf命令

  1. 输出类型:

    1. %ns:输出n个字符串
    2. %ni:输出n个整型
    3. %m.nf:输出m位整数,n位小数

  2. 实例:

    1. printf ‘%s’ 123
    2. printf ‘%s %s %s\n %s’ 1 2 3 4
    3. printf ‘%s’ $(cat testcut.txt)
    4. printf ‘%s\t %s\t %s\t %s\t’ $(cat testcut.txt)

3.awk命令：默认制表符和空格

  1. 形式:awk ‘条件1{动作1} 条件2{动作2}’ 文件名

  2. 实例1:awk ‘{printf $2”\t” $4”\n”}’ testcut.txt

    1. $2:输出第二列，$4输出第四列

  3. 实例2:awk’BEGIN{printf “start\n”} {printf $2”\t” $4”\n”} testcut.txt’

    1. BEGIN:最先执行后面{}中的指令

  4. 实例3:awk ‘BEGIN{FS=”:”}{printf $1”\t” $4”\n”} testcut.txt’

    1. {FS=”:”}:指定分隔符为”:”

  5. 实例4:awk ‘BEGIN{FS=”:”} END{print “123”} {printf $1”\t” $4”\n”} testcut.txt’

  6. 实例5:cat testcut.txt | grep -v Name | awk ‘$4 >= 80 {printf $2”\n”}’

    1. grep -v Name:查找不含Name字段的行

4. sed命令:仅对输出操作

  1. 形式:sed 选项 ‘动作’ 文件名

  2. 实例1:sed -n ‘2p’ testcut.txt

    1. -n:输出指定行(2p指第二行)

  3. 实例2:sed ‘2a hello’ testcut.txt

    1. 2a hello:在第二行后追加
    2. 2i hello:在第二行前追加

  4. 实例3:sed ‘2c no this people’ testcut.txt

5. 字符串处理命令

  1. 形式:sort 选项 文件名
    1. -f:忽略大小写
    2. -n:按数值型排序
    3. -r:反向排序
    4. -t:指定分隔符
